Блок 1 — Типы, операции, динамическая типизация, строки

1) Переменная и присваивание

  • Переменная — имя, которое хранит значение: x = 5.
  • Знак = — это “присвоить”, а не “равно”.
  • Код выполняется сверху вниз, переменная может перезаписываться.
x = 5
x = x + 2
print(x)  # 7

2) Типы данных (самые важные на старте)

  • int — целые числа: 3
  • float — дробные: 3.14
  • str — строки: "3"
  • bool — истина/ложь: True, False
print(type(3))      # int
print(type(3.0))    # float
print(type("3"))    # str
print(type(True))   # bool

3) Динамическая типизация (важнейшая идея)

В Python переменная не “закреплена” за одним типом. Это значит: сегодня x может быть числом, а через минуту — строкой. 
x = 10
print(type(x))   # int

x = "hello"
print(type(x))   # str

4) Операции и “ловушка” со строками

  • Для чисел: 2 + 3 = 5
  • Для строк: "2" + "3" = "23" (склейка)
  • "ha" * 3 = "hahaha"
print("2" + "3")   # "23"
print(2 + 3)       # 5
print("ha" * 3)    # "hahaha"

5) Базовые методы строк (минимум)

  • s.strip() — убрать пробелы по краям
  • s.lower(), s.upper() — регистр
  • s.split() — разбить на слова
  • len(s) — длина строки
s = "  Hello world  "
print(s.strip())          # "Hello world"
print(s.strip().lower())  # "hello world"
print("a b c".split())    # ["a", "b", "c"]
print(len("abc"))         # 3
Запомнить достаточно одного принципа: смотрим на тип и понимаем, что делает оператор/метод именно для этого типа.

Блок 2 — Списки, кортежи, словари + if/else

1) Список (list)

  • Список хранит много значений в порядке: [10, 20, 30]
  • Индексация: nums[0] — первый, nums[-1] — последний
  • Добавление: append, длина: len
nums = [10, 20, 30]
print(nums[0])     # 10
print(nums[-1])    # 30
nums.append(40)
print(nums)        # [10, 20, 30, 40]
print(len(nums))   # 4

2) Кортеж (tuple)

  • Кортеж похож на список, но обычно используется как “пара/тройка значений”: (x, y)
  • Удобно распаковывать: x, y = (3, 5)
p = (3, 5)
x, y = p
print(x, y)  # 3 5

3) Словарь (dict)

  • Хранит пары ключ → значение: {"name": "Маша", "age": 20}
  • Доступ: d["age"]
  • Безопасно: d.get("age") (вернёт None, если ключа нет)
person = {"name": "Маша", "age": 20}
print(person["age"])        # 20
print(person.get("city"))   # None

4) Простые ветвления if/else

if — это “если условие True, сделай одно, иначе — другое”.
Условие — это выражение, которое даёт True или False. 
x = int(input())
if x % 2 == 0:
    print("четное")
else:
    print("нечетное")

5) Самая важная привычка новичка

  • Если “что-то странно”: проверь тип через type()
  • Если KeyError: используй get()
  • Если IndexError: проверь len()
